# ShopPilot AI

AI-powered retail business operating system

## Problem

Small businesses struggle with billing, stock management, customer tracking, and business insights.

## Solution

ShopPilot AI automates:

- billing
- inventory
- customer intelligence
- AI insights
- voice commands
- invoice generation
- WhatsApp automation

## Features

- Smart Billing
- PDF Invoices
- Customer Intelligence
- Inventory Tracking
- AI Assistant
- Voice Commands
- Sales Analytics
- Low Stock Alerts
- Pending Payment Tracking
- Business Health Insights

## Tech Stack

- React
- Node.js
- Express
- MongoDB
- OpenAI API
- Twilio WhatsApp API
- Recharts
- TailwindCSS

## Demo Flow

1. Add customer
2. Create invoice
3. Download PDF
4. Send WhatsApp invoice
5. Track analytics
6. Ask AI business questions

## Why It Matters

Built for small retailers in India to digitize operations.
